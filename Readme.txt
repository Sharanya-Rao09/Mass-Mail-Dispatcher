==================================MASS EMAIL DISPATCHER WEBSITE ==========================================

The Mass-Mail Dispatcher project is a web-based application designed to simplify the process of sending mass emails to many recipients. A mass email dispatcher, email validation, and a user-friendly front-end website play a crucial role in effective email communication for businesses and organizations. This website has been developed using HTML5, CSS and JAVASCRIPT and uses EmailJS API to send the valid mails.

>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>>

Instructions to use the website(user instructions):

 1. Open the index.html file in your web browser.
 2. Upload a CSV file containing email addresses.
 3. The website will then sort the email addresses and separate the valid and invalid emails.
 4. Enter from user, email subject and body(message).
 5. Click on the send Emails button to dispatch the emails.
